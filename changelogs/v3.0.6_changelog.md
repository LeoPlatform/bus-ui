# Botmon v3.0.6 Changelog

## Improvements

### Event viewer: clarify “replay” (checkpoint reset + forward processing) [ES-2954](https://chb.atlassian.net/browse/ES-2954)

The CCW action on an event row **does not replay only that row**. It **sets the selected bot’s read checkpoint to the chosen event** and **runs processing from there forward**, including **later events on the same queue**.

- **Modal title:** “Reprocess from this event”
- **Primary button:** “Reset checkpoint & reprocess”
- **In-modal callout** (“How this works”) with short bullets describing checkpoint + forward processing
- **Confirm / toast copy** aligned with checkpoint semantics
- **CCW icon tooltip:** “Replay from this event” (short hint; full detail remains in the modal)

_Optional visuals:_ add PNGs on the **`changelog-screenshots`** branch under `changelogs/screenshots/`, then link here using the same GitHub raw URL pattern as **`v3.0.4_changelog.md`** (keeps binaries off **`master`**)._

---

### Page shell: `CustomJS` / `getTokenAdminFirstJQuery.js`

- **`views/index`:** load **`CustomJS`** (e.g. shared get-token script in test/prod) **only when** the value is a real `http` URL.
- Skips broken **`src="undefined"`** when **leo-cli** local dev does not substitute the parameter.

---

### Local development & publish (Node 20 + OpenSSL 3)

- **`npm start`:** applies **`test/process.js`** env before **leo-cli** (reduces Browserify / **leo-sdk** config errors during API bundling).
- **`npm run publish`** and **`npm run build`:** set **`NODE_OPTIONS=--openssl-legacy-provider`** for webpack/babel (avoids **`ERR_OSSL_EVP_UNSUPPORTED`** / digital envelope errors).
- **`publish:raw` / `build:raw` / `start:raw`:** call **leo-cli** directly without those wrappers.
- **README** expanded checklist and troubleshooting (Cognito, CustomJS, HMR, **`webpackPoll`**, publish tags).

---

### Live reload (local `leo-cli test`)

- **`ui/js/index.js`:** **`module.hot.accept`** + full page reload so webpack rebuilds refresh the tab (**leo-cli** uses **`webpack-hot-middleware`** with **`reload=false`**).
